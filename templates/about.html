What is motivate.me?

Motivate.me generates those motivational images that your mother likes to posts on facebook.


How does it work?

Simply, And badly. It fetches a random stock photo from Pexels, Sticks a random motivational quote on it from a list of about 300 motivational quotes, And then it shows it to you.


Could you be more specific?

This whole shit is a Flask app, When the index page is requested, It grabs a random image from Pexels using the Pexels API, Then it picks a quote from this file. Once the image and quote have been selected, It uses PIL to slap the quote on top of the image which is then saved to a file. Then it serves you index.html, Which contains the aforementioned image.


It's worth noting that the Pexels API is rate-limited to 200 requests per hour, And 20,000 requests per month. If that's exceeded, This app doesn't deal with it at all so it will probably just crash :)
